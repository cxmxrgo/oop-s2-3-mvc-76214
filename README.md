# VGC Student & Course Management System

## Prerequisites

- .NET SDK 10
- MySQL Server running locally

## Configuration
- `server=localhost;port=3306;database=vgc_student_course_m1;user=root;password=admin`

## Setup Steps
1. Open a terminal:
2. Restore packages:
   - `dotnet restore`
3. Apply migrations:
   - `dotnet ef database update`
4. Run the application:
   - `dotnet run`
